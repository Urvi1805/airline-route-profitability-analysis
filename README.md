# Airline Route Profitability & Network Optimization Analysis

## Overview

This project analyzes airline operational and financial data to evaluate route profitability, airport congestion, aircraft utilization, fuel costs, and flight delays. The objective is to generate data-driven recommendations for network optimization and operational efficiency using business analytics techniques.

## Objectives

- Analyze profitable and loss-making routes
- Evaluate airport congestion
- Measure aircraft utilization
- Analyze fuel expenditure
- Study flight delay patterns
- Develop interactive Tableau dashboards
- Recommend strategies for route optimization and cost reduction

## Tech Stack

- Tableau
- Python
- SQL
- Microsoft Excel
